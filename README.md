# 📘 Repositório do Curso de Fundamentos de IA e Análise de Dados

Este repositório contém os códigos, exercícios e bases de dados utilizados ao longo das aulas do curso de Fundamentos de IA e Análise de Dados. O objetivo do projeto é registrar o aprendizado prático em Python e suas principais bibliotecas.

---

## 🛠️ Conteúdo das Aulas

* **Introdução ao Python (`app.py`):**
  * Primeira introdução à linguagem Python.
  * Resolução de alguns exercícios básicos para ambientação e nivelamento.

* **Aula 02 (`aula02.ipynb`):**
  * Introdução aos operadores comparadores e estruturas condicionais/adicionais.

* **Aula 03 (`aula03.ipynb`):**
  * Primeira análise de dados utilizando uma base de dados real.
  * Introdução às bibliotecas essenciais: **Pandas** e **NumPy**.

* **Aula 04 (`aula04.ipynb`):**
  * Manipulação de tabelas e cruzamento de dados.
  * Introdução à biblioteca **Seaborn** para a criação de gráficos e análises visuais.

* **Aula 06 (`aula06.ipynb`):**
  * Notebook totalmente documentado e comentado.
  * Foco em visualização de dados, explorando diferentes tipos de gráficos com ênfase no uso de **Heatmaps** (Mapas de Calor).

* **Exercício de Fixação (`bibliotecas.ipynb`):**
  * Importação prática do **NumPy** e desenvolvimento de exercícios para fixar os conceitos da biblioteca.

---

## 📁 Bases de Dados (`/data`)

A pasta `/data` concentra todos os arquivos e tabelas utilizados durante as práticas:

* **`base_estudantes_treated.csv`:** Base de dados onde foi realizado o tratamento e a limpeza de dados incorretos de uma tabela anterior.
* **`dataset_com_total.csv`:** Base de dados manipulada para exibir valores totais agregados específicos.
* **Outros arquivos:** Contém os datasets originais (`dataset_aula02_exercicio_modafina.csv`, `dataset_aula03_student_habits_performance.csv`, `dataset_techshop_vendas.csv`) que servem de insumo para os exercícios.

---

> 💡 **Nota:** A partir da Aula 02, todas as atividades passaram a utilizar o **Jupyter Notebook (`.ipynb`)** para garantir uma leitura de código mais fluida, visual e organizada.
